# 自动部署项目到 GitHub Pages 的项目模板

已经配置好 GitHub Actions，推送代码到 master 分支就会自动部署项目到 GitHub Pages 生成预览页面

### 使用 Vue3 + Typescript + Vite

## 只需一步操作

只需在 GitHub 仓库 Setting > Pages > Source 选择 GitHub Actions
